UI Loading Strategy

How the session-specific UI interacts with the launcher core.

A

Complete UI Replacement

The session's HTML replaces the entire launcher view. The core Rust backend listens for specific id interactions (e.g., #playButton) to trigger actions like launching the game.

Interaction Example

<!-- Remote HTML from session -->
<div id="playButton">PLAY NOW</div>

// Rust (Tauri) handles the click automatically
// by binding to that ID in the loaded webview.
