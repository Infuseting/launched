Design: Core Architecture

A lightweight Tauri shell managing high-performance Rust services.

System Components
  • Tauri Frontend: A thin WebView shell that loads remote HTML.
    • Bridge Script: Injected into the WebView to bind IDs (#playButton, #sessionSwitcher, etc.) to Rust commands.
  • Rust Core Engine:
    • Session Manager: Fetches servers.json and manages session state.
    • Sync Service: Handles MD5-based file synchronization and syncDir cleanup.
    • Launch Service: Builds JVM arguments and manages the Java process (with/without logs).
    • Auth Manager: Implements the AuthStrategy pattern (Microsoft, SSO, etc.).
    • Storage Optimizer: Ensures shared versions are stored in .minecraft/versions and modpacks in isolated directories.

The "Bridge Script" is the key to Approach 1, making the remote HTML interactive without requiring the session creator to know Tauri's internals.
