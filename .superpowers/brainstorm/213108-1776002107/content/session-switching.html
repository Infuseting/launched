Session Switching & Bootstrapping

How users move between different modpack environments.

A

Session Switcher in Remote HTML

The session's own UI (from htmlPath) includes the buttons/UI to switch to other sessions. The Rust backend provides an API (e.g., window.__TAURI__.invoke('switch_session', { name: '...' })) for the HTML to call.

B

Core Launcher Overlay

The launcher has a "system" overlay or a specific keybinding/button (e.g., top-right corner) that opens a native or core-provided UI to switch sessions, regardless of the loaded HTML.
