Minecraft Launcher (Rust + Tauri)

Building a multi-session, high-performance launcher.

Core Features

  • Rust + Tauri: Lightweight and secure foundation.
  • Multi-Session: Each session can have its own UI/UX (loaded remotely).
  • Auto-Update: Stay current automatically.
  • Storage Optimized: Shared .minecraft/versions to save space.
  • Versatile Authentication: Strategy pattern for Microsoft, SSO, etc.
